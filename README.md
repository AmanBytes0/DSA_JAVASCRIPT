# DSA With JavaScript

This repository contains beginner-friendly Data Structures and Algorithms practice problems solved using JavaScript.

The goal of this repo is to build strong problem-solving fundamentals by practicing simple array problems, pattern printing, searching, and basic logic building.

## Topics Covered

- Arrays
- Loops
- Conditional logic
- Pattern printing
- Searching elements
- Finding largest and second largest values
- Removing duplicates
- Moving zeroes
- Two sum
- Basic counting problems

## How To Run

Make sure Node.js is installed on your system.

Run any JavaScript file using:

```bash
node filename.js
```

Example:

```bash
node largestelementinthearrrya.js
```

## Practice List

Some examples:
